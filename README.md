# NotaryTool

Phase 1 starter shell for the PA Compliance Utility MVP.

## What this is
- A Next.js frontend shell
- Prepared for Vercel deployment
- Prepared for Supabase environment variables
- Placeholder routes for Sanitize, Review, and Dashboard

## What this is not
- Not a journal
- Not a RON platform
- Not an approved solution provider product
- Not storing raw documents in the database

## Required environment variables
Create `.env.local` with:

```bash
NEXT_PUBLIC_SUPABASE_URL=https://ikhvtktvsvtfmzyrypto.supabase.co
NEXT_PUBLIC_SUPABASE_ANON_KEY=YOUR_SUPABASE_ANON_KEY
```

## Phase map
- Phase 1: skeleton and deployment
- Phase 2: real sanitize engine
- Phase 3: witnessing fee and dashboard
- Phase 4: desk sheet and launch
